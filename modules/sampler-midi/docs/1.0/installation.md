## Installation

```bash
# Using pnpm (recommended for monorepo)
pnpm add @audiocontrol/sampler-midi

# Using npm
npm install @audiocontrol/sampler-midi

# Using yarn
yarn add @audiocontrol/sampler-midi
```

### Dependencies

This package requires:

- `easymidi` - Default MIDI backend for Node.js
- `@audiocontrol/sampler-devices` - Akai device abstractions
- `@audiocontrol/sampler-lib` - Core utilities
